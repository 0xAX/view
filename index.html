What is it view.js?

View.js - is an html/javascript very very lightweight framework for building presentations. Lightweight and simplicity is the main point of view.js. Just include view.js file to your html file, fill content, write css design for your presentation as you choose and that's all, it's ready to use.

What is in?

  • css/content.css - content style (change it to your needs of your design)
  • css/layout.css - presentation layout (! do not edit this !)
  • js/view.js - main view.js functional (! do not edit this !)

How to use?

  1. Create an html file and include the following in the head section: css/layout.css(! do not edit this stylesheet !). Next add css/content.css for your own styling.
  2. Add the view.js file in your html file. Simply add the script tag right above the body tag, like so: script src="js/view.min.js".
  3. Mark up the html file with your content you want to be displayed in the presentation. Slides are created using the class slide, like so: div class="slide".
